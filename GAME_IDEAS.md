## 🔥 개발 예정 (우선순위순)

### 1. 스피드 카운팅 (Speed Counting) ⭐ 다음 개발 예정

> 탄토알/산도알 시리즈 '승객 세기' 미니게임에서 영감

**모티브**: 세가 탄토알 시리즈 - 개수 세기

**게임 규칙**:

1. 화면을 빠르게 지나가는 물체 개수 세기
2. 숫자패드로 정답 입력
3. 시간 제한 + 점점 빨라짐

**장점**: 기존 Speed Math 숫자패드 인프라 재활용 가능

**예상 개발 시간**: 1일

---

### 2. 아웃라이어 터치 (Odd One Out)

> 탄토알/산도알 시리즈 '삼각추' 미니게임에서 영감

**게임 규칙**:

1. N개의 도형 중 색/모양이 다른 1개 터치
2. 라운드마다 개수 증가 (9개 → 16개 → 25개)
3. 차이가 점점 미묘해짐

**장점**: 구현 간단, 무한 리플레이 가능

**예상 개발 시간**: 1일

---

### 3. 그림자 매칭 (Shadow Match)

> 탄토알/산도알 시리즈 '실루엣 맞추기' 미니게임에서 영감

**게임 규칙**:

1. 상단에 원본 이미지
2. 하단에 여러 실루엣 (회전/반전됨)
3. 원본과 일치하는 실루엣 터치

**장점**: 시각적으로 예쁨, 에셋 활용도 높음

**예상 개발 시간**: 2일 (에셋 포함)

## 📦 완료된 게임 상세 (아카이브)

### 매스 플라이트 (Math Flight)

> 종스크롤 러닝 액션 + **중간값 찾기** 두뇌 게임

**상태**: ✅ 완료 (`math-flight`)

**모티브**: 드래곤 플라이트 + 수학 퍼즐

**게임 규칙**:

1. 종스크롤 화면, 하단에 캐릭터 배치 (좌우 자유 이동)
2. 5개의 운석이 동시에 떨어짐 (각각 다른 숫자)
3. **충돌 판정**:
   - **가장 큰 수 / 가장 작은 수** 운석 → ❌ 라이프 손실
   - **중간 3개** 운석 → ✅ 점수 획득
   - **정확히 중간값** 운석 → ⭐ **2배 점수**
4. **핵심 전략**: 5개 숫자 중 최소/최대를 빠르게 파악하고 피하기!

**예시**:

```
운석: [12, 18, 25, 32, 38]
       ❌  ✅  ⭐  ✅  ❌
      실패 성공 2배! 성공 실패
```

**난이도 시스템** (숫자 범위: 0~50):

| 난이도 | 성공 범위 (±) | 실패 범위 (±) | 예시 (중간값 25)     |
| ------ | ------------- | ------------- | -------------------- |
| 초반   | 5~8           | 10~15         | [8, 18, 25, 32, 42]  |
| 중반   | 4~7           | 7~11          | [15, 19, 25, 31, 35] |
| 후반   | 3~5           | 4~8           | [18, 21, 25, 29, 32] |

- 초반: 숫자 간격 넓음 → 쉬움
- 후반: 숫자가 거의 연속 → 빠른 판단 필요!
- 낙하 속도도 점진적 증가

**기술 고려사항**:

- 터치/드래그 → 캐릭터 자유 이동 (드래곤 플라이트 스타일)
- 운석 생성 알고리즘 (난이도별 분포 조절)
- 거리 기반 충돌 감지
- 무한 러닝 (라이프 소진까지)

**예상 개발 시간**: 2~3일 (밸런스 조정 포함)

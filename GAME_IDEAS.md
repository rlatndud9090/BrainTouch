## 🔥 개발 예정 (우선순위순)

### 1. 블록셈 (Block Sum) ⭐ 다음 개발 예정

> 일본 전통놀이 '다루마 오토시'에서 영감을 받은 덧셈 퍼즐

**모티브**: 말랑말랑 두뇌학원 - 덧셈 블록

**게임 규칙**:

1. 숫자가 쓰인 블록 4~6개가 탑처럼 세로로 쌓여 등장
2. 목표 숫자가 제시됨
3. 블록을 터치 후 **왼쪽 스와이프**로 제거
4. **남은 블록의 합 = 목표 숫자**가 되면 성공!
5. 블록을 적게 제거할수록 높은 점수
6. 목표 달성 불가능 상태 → 실패 처리
7. **60초** 제한시간 내 최대한 많이 클리어

**난이도 시스템**:
| 난이도 | 블록 개수 | 숫자 범위 | 승급 조건 |
|--------|----------|----------|----------|
| 하 (Easy) | 4개 | 1~9 | 3회 성공 시 → 중 |
| 중 (Normal) | 5개 | 1~12 | 3회 성공 시 → 상 |
| 상 (Hard) | 6개 | 1~15 | 유지 |

**기술 고려사항**:

- 스와이프 제스처 감지 (Phaser input)
- 블록 제거 시 위 블록들 자연스럽게 낙하 애니메이션
- "달성 가능 여부" 검증: 조합 수가 적으므로 (2^6 = 64) 병목 없음
- 물리 시뮬레이션 필요 없음 (단순 tween으로 처리)

**UI 참고** (말랑말랑 두뇌학원):

- 블록이 세로로 쌓임
- 목표 숫자가 오른쪽에 크게 표시
- 제한 시간 상단
- 점수 우상단

**예상 개발 시간**: 1~2일

### 2. 매스 플라이트 (Math Flight) 🚧 개발 중

> 종스크롤 러닝 액션 + **중간값 찾기** 두뇌 게임

**모티브**: 드래곤 플라이트 + 수학 퍼즐

**게임 규칙**:

1. 종스크롤 화면, 하단에 캐릭터 배치 (좌우 자유 이동)
2. 5개의 운석이 동시에 떨어짐 (각각 다른 숫자)
3. **충돌 판정**:
   - **가장 큰 수 / 가장 작은 수** 운석 → ❌ 라이프 손실
   - **중간 3개** 운석 → ✅ 점수 획득
   - **정확히 중간값** 운석 → ⭐ **2배 점수**
4. **핵심 전략**: 5개 숫자 중 최소/최대를 빠르게 파악하고 피하기!

**예시**:

```
운석: [12, 18, 25, 32, 38]
       ❌  ✅  ⭐  ✅  ❌
      실패 성공 2배! 성공 실패
```

**난이도 시스템** (숫자 범위: 0~50):

| 난이도 | 성공 범위 (±) | 실패 범위 (±) | 예시 (중간값 25)     |
| ------ | ------------- | ------------- | -------------------- |
| 초반   | 5~8           | 10~15         | [8, 18, 25, 32, 42]  |
| 중반   | 4~7           | 7~11          | [15, 19, 25, 31, 35] |
| 후반   | 3~5           | 4~8           | [18, 21, 25, 29, 32] |

- 초반: 숫자 간격 넓음 → 쉬움
- 후반: 숫자가 거의 연속 → 빠른 판단 필요!
- 낙하 속도도 점진적 증가

**기술 고려사항**:

- 터치/드래그 → 캐릭터 자유 이동 (드래곤 플라이트 스타일)
- 운석 생성 알고리즘 (난이도별 분포 조절)
- 거리 기반 충돌 감지
- 무한 러닝 (라이프 소진까지)

**예상 개발 시간**: 2~3일 (밸런스 조정 포함)

# 중간값 비행 스프라이트 프롬프트

GPT 웹 이미지 생성에 바로 붙여 넣어 사용할 프롬프트 모음입니다.

## 공통 생성 규칙

- 출력 형식: `PNG`
- 배경: `투명 배경`
- 구도: 단일 오브젝트 중앙 배치
- 스타일: 동글동글하고 귀여운 캐주얼 2.5D 게임 스프라이트
- 금지: 텍스트, 숫자, 워터마크, 로고, 배경 소품
- 권장 해상도: `1024x1024`

## 파일명 매핑

- `ship_player.png` : 플레이어 우주선
- `meteor_core.png` : 운석 본체
- `meteor_trail_flame.png` : 운석 비행 불꽃
- `meteor_split_hit.png` : 운석 피격 분리 스프라이트

## 1) 플레이어 우주선 (`ship_player.png`)

```text
탑다운 시점의 귀여운 소형 우주선 스프라이트. 기본 형태는 삼각형이지만 모서리는 둥글고 말랑한 느낌. 본체 색상은 밝은 시안(#00d4ff), 하단 엔진 노즐과 포인트는 주황(#ff6b35). 작은 화면에서도 식별되도록 단순하고 또렷한 실루엣, 과도한 디테일 없이 게임 UI 친화적인 스타일.
```

## 2) 운석 본체 (`meteor_core.png`)

```text
둥근 운석 돌 스프라이트. 살짝 납작한 원형 실루엣, 표면에 작은 크레이터와 짧은 균열 텍스처, 색상은 푸른 회색(#4a6fa5 계열). 너무 사실적인 질감 대신 귀엽고 카툰스러운 2.5D 음영으로 표현. 중앙 오브젝트만 깔끔하게.
```

## 3) 운석 비행 불꽃 (`meteor_trail_flame.png`)

```text
운석 뒤에 붙는 불꽃/트레일 이펙트 스프라이트. 물방울처럼 길게 뻗는 형태, 중심은 주황~노랑 그라데이션, 외곽은 시안빛 글로우. 속도감이 느껴지지만 과하게 날카롭지 않고 둥근 느낌의 캐주얼 FX 스타일.
```

## 4) 운석 피격 분리 (`meteor_split_hit.png`)

```text
운석이 충돌로 반으로 갈라진 순간의 스프라이트. 좌우 두 파편이 조금 벌어진 형태, 갈라진 단면은 둔한 톱니 느낌. 내부 단면에 약한 주황빛 열기 표현, 작은 파편 2~3개 포함. 과한 폭발 없이 히트 연출용으로 읽기 쉬운 구성.
```

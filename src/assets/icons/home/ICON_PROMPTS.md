# BrainTouch 홈 아이콘 프롬프트

아래 프롬프트를 GPT 웹 이미지 생성에 각각 1개씩 넣어 생성하면 됩니다.

## 공통 생성 규칙

- 정사각형 아이콘, 1:1 비율
- 동글동글하고 귀여운 캐주얼 게임 아이콘 스타일
- 살짝 3D 느낌의 말랑한 스티커 렌더링
- 밝고 선명한 색감, 높은 가독성
- 배경은 투명 PNG (불가능하면 단색 배경 + 피사체 분리 쉬운 구성)
- 중앙 배치, 작은 화면에서도 인식 가능
- 텍스트, 글자, 숫자 UI, 워터마크, 로고 금지

## 1) 전체 게임 아이콘 (브레인 터치 - 계산편)

```text
모바일 두뇌게임 앱 아이콘. 둥근 귀여운 뇌 캐릭터와 작은 주판(또는 숫자 블록) 요소가 함께 있는 구성. 계산과 터치 반응을 상징하는 부드러운 빛 효과를 추가. 동글동글한 3D 스티커 스타일, 캐주얼 게임 느낌, 밝고 친근한 색감, 높은 대비, 중앙 배치, 아이콘용 심플한 형태. 배경은 투명 PNG. 텍스트/워터마크/로고 없음.
```

## 2) 몸풀기 터치 아이콘

```text
숫자 타겟을 손가락으로 터치하는 순간을 표현한 게임 아이콘. 동그란 숫자 타일 하나와 귀여운 손가락 포인터, 터치 파동 이펙트. 반응속도 게임 느낌이 나도록 역동적이지만 단순한 구성. 동글동글한 3D 스티커 스타일, 밝고 선명한 색, 중앙 배치, 작은 크기에서도 잘 보이게. 배경은 투명 PNG. 텍스트/워터마크/로고 없음.
```

## 3) 스피드 계산 아이콘

```text
귀여운 스톱워치와 수식 힌트 요소(플러스/마이너스/곱하기 모양 아이콘)를 조합한 게임 아이콘. 빠르게 계산하는 느낌이 나도록 가벼운 속도선 효과. 동글동글한 3D 스티커 스타일, 캐주얼하고 친근한 분위기, 중앙 배치, 고대비 색상. 배경은 투명 PNG. 텍스트/워터마크/로고 없음.
```

## 4) 중간값 비행 아이콘

```text
귀여운 작은 로켓이 숫자가 적힌 둥근 운석 사이를 날아가는 게임 아이콘. 회피 액션 + 계산 퍼즐 느낌이 함께 보이게 구성. 위험하지 않고 밝은 톤의 캐주얼 스타일. 동글동글한 3D 스티커 렌더, 중앙 배치, 선명한 색감, 아이콘에 최적화된 단순한 실루엣. 배경은 투명 PNG. 텍스트/워터마크/로고 없음.
```

## 5) 블록셈 아이콘

```text
숫자가 적힌 말랑한 큐브 블록이 층층이 쌓여 있고, 하나의 블록이 스와이프로 빠지는 순간을 표현한 퍼즐 게임 아이콘. 합계를 맞추는 느낌이 나도록 블록 숫자 대비를 또렷하게. 동글동글하고 귀여운 3D 스티커 스타일, 중앙 배치, 밝은 색감, 작은 크기에서도 인식 쉬운 구성. 배경은 투명 PNG. 텍스트/워터마크/로고 없음.
```

```text
크기가 다른 귀여운 풍선 3~4개가 순서대로 배치된 게임 아이콘. 숫자 순서 맞추기 느낌을 주되 복잡하지 않게 단순한 구성. 말랑하고 동글동글한 3D 스티커 스타일, 밝고 화사한 색감, 중앙 배치, 아이콘 가독성 우선. 배경은 투명 PNG. 텍스트/워터마크/로고 없음.
```

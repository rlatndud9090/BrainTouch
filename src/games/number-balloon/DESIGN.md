# 🎈 숫자풍선 (Number Balloon) 설계 문서

> 말랑말랑 두뇌학원의 '숫자풍선' 게임에서 영감을 받은 순서 맞추기 게임

---

## 📌 게임 개요

| 항목          | 내용                                 |
| ------------- | ------------------------------------ |
| **게임 ID**   | `number-balloon`                     |
| **장르**      | 퍼즐 / 두뇌 게임                     |
| **조작**      | 터치 (풍선 탭)                       |
| **게임 방식** | 무한 모드 (3번 실패 시 종료)         |
| **목표**      | 작은 숫자부터 순서대로 풍선 터뜨리기 |

---

## 🎮 게임 규칙

### 기본 규칙

1. 화면에 숫자가 적힌 풍선 N개 등장
2. **작은 숫자부터 순서대로** 터치하여 터뜨림
3. 순서대로 터뜨리면 점수 획득
4. 틀리면 ❤️ 라이프 -1
5. **3번 실패 시 게임 오버**
6. 라운드 클리어 시 다음 라운드로 (난이도 상승)

### 점수 시스템

```
점수 = 10 × 현재 라운드

예시: Round 5에서 풍선 터뜨림 → +50점
```

---

## 📊 난이도 시스템 (핵심!)

### 형님 알고리즘: 정렬 기반 매칭 + Index 교환

```
1. 풍선 크기 배열 랜덤 생성 → 오름차순 정렬
2. 숫자 배열 랜덤 생성 → 오름차순 정렬
3. 기본 매칭: 같은 index (작은 숫자 = 작은 풍선)
4. 난이도에 따라 index 교환 쌍 생성
   - 교환 쌍 개수 증가
   - 교환 가능한 최대 index 차이 증가
```

### 난이도 파라미터

| 라운드 | 풍선 개수 | 교환 쌍 개수 | 최대 index 차이 | 숫자 범위 |
| ------ | --------- | ------------ | --------------- | --------- |
| 1~2    | 5개       | 0~1개        | 1~2             | 0~9       |
| 3~4    | 5~6개     | 1개          | 2               | 0~15      |
| 5~6    | 6~7개     | 1~2개        | 2~3             | 0~20      |
| 7+     | 7~9개     | 2~3개        | 3~4             | 0~30+     |

### 예시

```
Round 1 (쉬움):
크기: [40, 50, 60, 70, 80] (정렬됨)
숫자: [2, 5, 8, 12, 15] (정렬됨)
→ 2=40px, 5=50px, 8=60px... (직관적)

Round 7 (어려움):
크기: [40, 50, 60, 70, 80] (정렬됨)
숫자: [2, 5, 8, 12, 15] (정렬됨)
index 0↔3 교환 → 크기: [70, 50, 60, 40, 80]
→ 2=70px(큰 풍선!), 12=40px(작은 풍선!) → 헷갈림!
```

## 🔧 기술 구현

### 파일 구조

```
src/games/number-balloon/
├── config.ts                 # Phaser 게임 설정
├── DESIGN.md                 # 이 문서
├── scenes/
│   ├── GameScene.ts          # 메인 게임 씬
│   └── ResultScene.ts        # 결과 화면
└── utils/
    └── BalloonGenerator.ts   # 풍선 생성 알고리즘 (핵심)
```

### BalloonGenerator 핵심 로직

```typescript
// 1. 크기/숫자 배열 생성 후 정렬
sizes.sort((a, b) => a - b);
numbers.sort((a, b) => a - b);

// 2. 교환 쌍 생성
const swapPairs = generateSwapPairs(count, swapCount, maxIndexDiff);

// 3. 크기 배열에서 index 교환
for (const [i, j] of swapPairs) {
  [sizes[i], sizes[j]] = [sizes[j], sizes[i]];
}

// 결과: 작은 숫자가 큰 풍선에, 큰 숫자가 작은 풍선에!
```

### 풍선 배치

- 겹치지 않게 랜덤 배치
- 충돌 감지로 위치 조정
- 흔들리는 애니메이션 (Tween)

## 🎨 디자인 가이드

### 색상

- **배경**: 하늘색 그라데이션 (#87CEEB → #B0E0E6)
- **풍선**: 핑크, 하늘, 주황, 초록, 빨강, 노랑, 보라, 민트
- **텍스트**: 흰색 + 검정 외곽선

### 풍선 스프라이트

```
     ___
   /     \    ← 원형 본체
   |  7  |    ← 숫자 (크기에 비례)
   \_____/
     |        ← 꼭지
     |        ← 끈
```

### 애니메이션

| 이벤트        | 애니메이션                       |
| ------------- | -------------------------------- |
| 대기          | 살짝 흔들림 (Sine.easeInOut)     |
| 정답 터치     | 터지는 효과 + 파티클 + 점수 팝업 |
| 오답 터치     | 화면 흔들림 + ❌ 표시            |
| 라운드 클리어 | 🎉 Clear! 텍스트 팝업            |

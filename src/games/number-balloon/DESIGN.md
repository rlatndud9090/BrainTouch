# 🎈 숫자풍선 (Number Balloon) 설계 문서

> 말랑말랑 두뇌학원의 '숫자풍선' 게임에서 영감을 받은 순서 맞추기 게임

---

## 📌 게임 개요

| 항목          | 내용                                 |
| ------------- | ------------------------------------ |
| **게임 ID**   | `number-balloon`                     |
| **장르**      | 퍼즐 / 두뇌 게임                     |
| **조작**      | 터치 (풍선 탭)                       |
| **게임 방식** | 무한 모드 (3번 실패 시 종료)         |
| **제한 시간** | 라운드별 6초 → 5초 → 4초            |
| **목표**      | 작은 숫자부터 순서대로 풍선 터뜨리기 |

---

## 🎮 게임 규칙

### 기본 규칙

1. 화면에 숫자가 적힌 풍선 N개 등장
2. **작은 숫자부터 순서대로** 터치하여 터뜨림
3. 순서대로 터뜨리면 점수 획득
4. 틀리면 ❤️ 라이프 -1
5. **3번 실패 시 게임 오버**
6. 라운드 클리어 시 다음 라운드로 (난이도 상승)

### 점수 시스템

```
점수 = 10 × 현재 라운드

예시: Round 5에서 풍선 터뜨림 → +50점
```

## 📊 난이도 시스템 (핵심!)

### 형님 알고리즘: 정렬 기반 매칭 + Index 교환

```
1. 풍선 크기 배열 랜덤 생성 → 오름차순 정렬
2. 숫자 배열 랜덤 생성 → 오름차순 정렬
3. 기본 매칭: 같은 index (작은 숫자 = 작은 풍선)
4. 난이도에 따라 index 교환 쌍 생성
   - 교환 쌍 개수 증가
   - 교환 가능한 최대 index 차이 증가
```

### 난이도 파라미터

현재 구현은 `utils/DifficultyDirector.ts`를 기준으로 세 축을 조합합니다.

#### 1. 풍선 부하 축 (`balloonLoad`)

| 레벨 | 풍선 개수 | 풍선 크기 범위 |
| ---- | --------- | -------------- |
| 0    | 5         | 52~74          |
| 1    | 6~7       | 48~70          |
| 2    | 8~9       | 44~64          |

#### 2. 매핑 혼란 축 (`mappingChaos`)

| 레벨 | 교환 쌍 개수 | 최대 index 차이 |
| ---- | ------------ | --------------- |
| 0    | 0            | 0               |
| 1    | 1            | 2               |
| 2    | 1            | 3               |
| 3    | 2            | 4               |
| 4    | 3            | 6               |
| 5    | 4            | 8               |

#### 3. 숫자 인지 축 (`numberCognition`)

| 레벨 | 숫자 범위 |
| ---- | --------- |
| 0    | 0~20      |
| 1    | 0~35      |
| 2    | 5~50      |
| 3    | 10~70     |
| 4    | 20~90     |
| 5    | 30~99     |

#### 시간 레벨

- 라운드 구간에 따라 `6초 → 5초 → 4초`
- 시간 레벨은 `1~24라운드: 6초`, `25~48라운드: 5초`, `49라운드+: 4초`로 24라운드 간격 상승
- 풍선 개수 축은 매 12라운드마다 오르는 것이 아니라 `1~12라운드: 5개`, `13~36라운드: 6개`, `37라운드+: 7개`로 13라운드와 37라운드에서만 증가

#### 승급/하향 규칙

- 성공 시 `mappingChaos` 또는 `numberCognition` 축이 점진적으로 상승
- 잘못된 터치 또는 시간초과 시 두 축 중 하나가 1단계 하향
- `balloonLoad` 축은 라운드 진행에 맞춰 자동 적용

### 예시

```
Round 1 (쉬움):
크기: [40, 50, 60, 70, 80] (정렬됨)
숫자: [2, 5, 8, 12, 15] (정렬됨)
→ 2=40px, 5=50px, 8=60px... (직관적)

Round 7 (어려움):
크기: [40, 50, 60, 70, 80] (정렬됨)
숫자: [2, 5, 8, 12, 15] (정렬됨)
index 0↔3 교환 → 크기: [70, 50, 60, 40, 80]
→ 2=70px(큰 풍선!), 12=40px(작은 풍선!) → 헷갈림!
```

```
    └── DifficultyDirector.ts # 축 기반 난이도/시간 제어
```

### BalloonGenerator 핵심 로직

```typescript
// 1. 크기/숫자 배열 생성 후 정렬
sizes.sort((a, b) => a - b);
numbers.sort((a, b) => a - b);

// 2. 교환 쌍 생성
const swapPairs = generateSwapPairs(count, swapCount, maxIndexDiff);

// 3. 크기 배열에서 index 교환
for (const [i, j] of swapPairs) {
  [sizes[i], sizes[j]] = [sizes[j], sizes[i]];
}

// 결과: 작은 숫자가 큰 풍선에, 큰 숫자가 작은 풍선에!
```

### 풍선 배치

- 겹치지 않게 랜덤 배치
- 충돌 감지로 위치 조정
- 흔들리는 애니메이션 (Tween)

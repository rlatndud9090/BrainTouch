# AGENTS

이 문서는 저장소에서 **실제로 확인된** 워크플로/명령어만 최소 요약합니다.

## 개발 명령어 (확인됨)

- `npm install`
- `npm run dev` (http://localhost:3000)
- `npm run build`
- `npm run preview`

## Git 워크플로 (CONVENTION 기준)

- 브랜치: `feature/*`, `fix/*`, `refactor/*`
- 커밋 포맷: `<type>(<scope>): <subject>`
